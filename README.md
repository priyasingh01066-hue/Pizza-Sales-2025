# Pizza-Sales(Jan2025-Dec2025)
This dashboard provides an overview of pizza sales performance from January 2025 to December 2025. It analyzes key business metrics, including Total Revenue, Total Orders, Total Pizzas Sold, Average Order Value, and Average Pizzas per Order. The report also highlights sales trends by day, month, pizza category, and pizza size, helping identify peak sales periods and customer preferences. These insights support better decision-making for inventory planning, marketing strategies, and overall business performance.

## Tools Used
* Power BI – Used for data extraction and dashboard development.
* Power Query – Used for data cleaning and transformation.
* DAX – Used to create calculated columns and measures.
* Data Visualization

## Key Insights
1. Total Revenue: 817.86K
2. Total Orders: 21,350
3. Total Pizzas Sold: 49,574
4. Top Category: Classic
5. Most Preferred Size: Large
6. Peak Month: July
7. Busiest Day: Thursday
